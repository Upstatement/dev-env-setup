# Upstatement Development Environment Setup

## Getting Started

1. Clone the repo

  ```sh
  git clone git@github.com:Upstatement/dev-env-setup.git && cd dev-env-setup
  ```

1. Run the install script

  ```sh
  sh install.sh
  ```
